# Neo-Brutalism vs. Brutalism: Quick Reference

## The Key Distinction

**This project showcases NEO-BRUTALISM, not pure brutalism.**

## What's the Difference?

### BRUTALISM (Pure/Architectural)
- **Origin**: 1950s-1970s architecture (Le Corbusier, raw concrete)
- **Digital example**: Craigslist, Drudge Report
- **Aesthetic**: Raw HTML, minimal CSS, unfinished, harsh
- **Colors**: Minimal (black, white, gray only)
- **Philosophy**: Truth to materials, honest function
- **Vibe**: Accidentally ugly or intentionally stripped down

### NEO-BRUTALISM (Digital Evolution)
- **Origin**: 2010s-2020s web design + 90s nostalgia
- **Digital examples**: Figma, Gumroad, Balenciaga
- **Aesthetic**: Structured but raw, blocky, geometric
- **Colors**: Bold primaries, bright colors, high contrast
- **Philosophy**: Brutalist honesty + 90s playfulness + modern UX
- **Vibe**: Intentionally bold, deliberately designed to look unpolished

## The Fusion Formula

```
NEO-BRUTALISM = 
  Brutalist Architecture (raw, honest, exposed structure)
  + 
  90s Web Design (blocky layouts, bright colors, simple HTML vibes)
  + 
  Modern UX (accessibility, usability, responsive)
```

## Why It Matters

### If this were PURE BRUTALISM:
- ❌ Only black/white/gray
- ❌ Plain HTML, minimal styling
- ❌ No structured layout
- ❌ Like Craigslist (unchanged since 1995)

### As NEO-BRUTALISM:
- ✅ Black + White + Bright Yellow
- ✅ Thick borders, hard shadows, geometric shapes
- ✅ Blocky but ordered layout
- ✅ Like Figma's rebrand (bold, colorful, structured)

## Visual Characteristics Comparison

| Feature | Brutalism | Neo-Brutalism (Our Site) |
|---------|-----------|-------------------------|
| **Colors** | Minimal (B/W/Gray) | Bold primaries (B/W/Yellow) |
| **Borders** | Rare or thin | Thick (4-5px), everywhere |
| **Shadows** | None | Hard, offset (8px) |
| **Layout** | Raw HTML | Blocky, geometric, CSS Grid |
| **Typography** | Plain system fonts | Bold, ALL CAPS, chunky |
| **Spacing** | Minimal | Balanced (not too tight) |
| **Structure** | Chaotic/minimal | Orderly but raw |
| **Nostalgia** | None (or 60s concrete) | 90s web (Windows 98 vibes) |

## Historical Timeline Distinction

### Brutalism Path
1. **1950s**: Brutalist architecture emerges
2. **1995**: Craigslist launches (accidental brutalism)
3. **2000s**: Web moves away from plain HTML
4. **2010s**: Some designers return to raw HTML aesthetics

### Neo-Brutalism Path
1. **1950s**: Brutalist architecture influences design philosophy
2. **1995-2000**: Early web has bold colors, thick borders, simple layouts
3. **2010s**: Designers react to over-polished interfaces
4. **2016+**: Brutalist PHILOSOPHY + 90s WEB AESTHETICS = Neo-Brutalism
5. **2020s**: Brands adopt (Figma, Gumroad)

## Key Neo-Brutalist Elements in Our Site

### From Brutalism:
- ✅ Exposed grid structure
- ✅ Honest, visible elements
- ✅ No subtle gradients
- ✅ Raw aesthetic

### From 90s Web:
- ✅ Bright yellow accent color
- ✅ Thick black borders (like old frames)
- ✅ Blocky, compartmentalized sections
- ✅ Hard drop shadows
- ✅ Web 1.0 simplicity

### Modern Polish:
- ✅ Responsive design
- ✅ Accessibility (WCAG contrast ratios)
- ✅ Clear hierarchy
- ✅ Usable navigation

## Brand Examples Clarification

### Pure Brutalism
- **Craigslist**: Unchanged HTML since 1995
- **Drudge Report**: Plain text, minimal styling
- **Early personal websites**: Raw HTML

### Neo-Brutalism
- **Figma**: Bold rebrand with thick borders, bright colors
- **Gumroad**: Chunky buttons, hard shadows, yellow accents
- **Balenciaga digital**: Fashion brand with brutalist-inspired web
- **Tony's Chocolonely**: Playful neo-brutalist e-commerce

## Nielsen Norman Group's Definition

From their research (April 2025):

> "Neobrutalism, an evolution of traditional brutalism, is a visual-design trend defined by high contrast, blocky layouts, bold colors, thick borders, and 'unpolished' elements."

Key phrase: **"evolution of traditional brutalism"**

## The Essence

**Brutalism** = Raw honesty (sometimes to a fault)  
**Neo-Brutalism** = Raw honesty + Intentional design + Colorful personality

Our site is firmly in the **NEO-BRUTALISM** category.
