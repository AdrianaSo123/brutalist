UNDERSTANDING THE STYLE

PHILOSOPHY, PRINCIPLES & PRACTICE

WHAT MAKES NEO-BRUTALISM AUTHENTIC?

Neo-Brutalism is not about being accidentally ugly or poorly designed. It's a deliberate rejection of over-refinement in favor of raw honesty. Understanding the difference between authentic Brutalism and simply bad design is crucial.

CORE PHILOSOPHY

01

TRUTH TO MATERIALS

Architectural: Show the concrete. Don't hide structure behind decorative facades. Let materials be themselves.

Digital: Show the grid. Expose the structure. Make borders visible. Don't hide the underlying system.

IN PRACTICE:

✓ Thick visible borders around sections

✓ Exposed grid lines

✓ Hard box shadows (not subtle blur)

✗ Seamless blending

✗ Invisible transitions

02

HONEST FUNCTION

Architectural: Form follows function. Every element serves a purpose. No decoration for decoration's sake.

Digital: Every element has a job. No decorative flourishes. Remove anything non-essential.

IN PRACTICE:

✓ Buttons clearly look clickable

✓ Navigation is obvious

✓ Hierarchy through size, not subtlety

✗ Decorative icons

✗ Unnecessary animations

03

ANTI-CORPORATE

Architectural: Reject ornamental tradition. Challenge established aesthetics. Stand against commercial polish.

Digital: Deliberately rough. Reject sleek interfaces. Challenge over-designed corporate sites.

IN PRACTICE:

✓ Purposefully unrefined

✓ Stark, high contrast

✓ No smooth animations

✗ Gradient buttons

✗ Subtle shadow depths

04

RAW AESTHETICS

Architectural: Exposed concrete. Visible pipes and ducts. Unfinished surfaces. Honest materials.

Digital: System fonts. Thick borders. Hard shadows. Sharp corners. Tight spacing.

IN PRACTICE:

✓ Arial, Helvetica, Courier

✓ 90° angles only (no border-radius)

✓ Black/white dominance

✗ Custom webfonts

✗ Rounded corners

DESIGN CHARACTERISTICS IN DEPTH

COLOR PALETTE

DO:

  • Use stark black (#000000) and white (#FFFFFF)
  • Add bright accent colors sparingly (yellow #FFFF00, red #FF0000)
  • Maintain high contrast always
  • Use pure colors straight from picker

DON'T:

  • Use gradients or color transitions
  • Apply subtle pastels or muted tones
  • Create complex color harmonies
  • Use transparency effects

TYPOGRAPHY

DO:

  • System fonts (Arial, Helvetica, Courier)
  • ALL CAPS for emphasis
  • Tight line-height (1.2-1.4)
  • Extreme size contrast (huge vs. small)
  • Bold, thick underlines

DON'T:

  • Custom webfonts (they're corporate bloat)
  • Serif fonts (too refined)
  • Generous line-height (too airy)
  • Subtle size variations
  • Refined letter-spacing adjustments

LAYOUT & GRID

DO:

  • Expose grid lines visibly
  • Use thick borders (4-5px) around sections
  • Create boxed, compartmentalized areas
  • Make structure obvious
  • Allow slight intentional misalignment

DON'T:

  • Hide the grid structure
  • Use subtle or invisible borders
  • Create seamless flowing sections
  • Apply rounded corners (border-radius)
  • Make symmetry subtle

SPACING

DO:

  • Keep spacing tight and packed
  • Use minimal padding (content near edges)
  • Create intentional tension
  • Break rhythm on purpose

DON'T:

  • Use generous white space (too Swiss)
  • Add excessive padding
  • Make layouts airy and breathable
  • Follow consistent spacing rules

COMMON MISTAKES TO AVOID

MISTAKE #1

Making It Too Pretty

Wrong: Subtle shadows, rounded corners, smooth gradients

Right: Hard shadows, 90° angles, flat colors

If it looks "nice," you're doing it wrong. Neo-Brutalism rejects refinement.

MISTAKE #2

Using Too Many Colors

Wrong: Full rainbow palette, pastels, muted tones

Right: Black, white, one or two bright accents

Brutalism is stark and high-contrast, not colorful and playful.

MISTAKE #3

Generous White Space

Wrong: Lots of padding, airy layouts, generous margins

Right: Tight, packed, intentionally cramped

Swiss design is generous. Brutalism reacts against that.

MISTAKE #4

Smooth Animations

Wrong: Fade transitions, ease curves, smooth hovers

Right: Instant state changes, hard snaps

Smoothness = polish = corporate. Brutalism is abrupt and honest.

UX BEST PRACTICES FOR BRUTALISM

Bold aesthetics must balance with usability. Based on UX research, here's how to make brutalism functional:

01

USABILITY FIRST

Clear buttons, readable text, ample whitespace. Bold aesthetic shouldn't compromise intuitive navigation.

02

CONTRAST RATIOS

Bold colors must meet WCAG standards. Black on white: 21:1. Yellow accents: 10.4:1. Always accessible.

03

LIMITED PALETTE

2-3 high-contrast colors maximum. Prevents overwhelming users while maintaining visual impact.

04

READABILITY

Bold headlines, clean body text. System fonts ensure legibility across all devices and contexts.

05

STRATEGIC WHITESPACE

24-32px margins offset bold elements. Creates breathing room despite tight brutalist philosophy.

06

CLEAR INTERACTIONS

Obvious hover states, instant feedback. No smooth transitions but clear state changes.

Source: Nielsen Norman Group UX Research
